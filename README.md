# Finance-Dashboard-UI
A clean and interactive finance dashboard to track income, expenses, and spending insights.  This project focuses on frontend design, user experience, and state handling without relying on a backend.

📌 Overview

This dashboard allows users to:

View financial summary (balance, income, expenses)
Analyze spending patterns using charts
Explore and filter transactions
Switch between roles (Viewer / Admin)
Gain useful financial insights
✨ Features
📊 Dashboard Overview
Summary cards (Total Balance, Income, Expenses)
Time-based visualization (trend chart)
Category-based visualization (spending breakdown)
📋 Transactions
View transaction list with:
Date
Amount
Category
Type (Income/Expense)
Search and filter functionality
Sorting support
🔐 Role-Based UI
Viewer
Can only view data
Admin
Can add/edit transactions (UI simulation)
💡 Insights Section
Highest spending category
Monthly comparison
Basic financial observations
🧠 State Management

The application uses centralized state handling with JavaScript variables.

State includes:

Transactions data
Active filters
Selected user role

Whenever state changes, the UI re-renders to reflect updated data.
